# **App Name**: SkillSwap

## Core Features:

- User Profiles: Create and manage profiles with 'Have' and 'Want' skill lists.
- Skill Tag System: Tag skills to categorize and filter user expertise.
- Peer Matching System: Algorithm to match users based on complementary 'Have' and 'Want' skills.
- Search & Filter: Tools to search for peers and filter skills, integrating the list of skill tags to easily find good learning partners
- Connection Requests: Send and manage connection requests to potential study partners.
- Basic Chat System: Real-time chat for connected users to discuss and coordinate learning sessions.
- Dashboard: AI-driven suggestions for optimal skill pairings.

## Style Guidelines:

- Primary color: A saturated, vibrant blue (#29ABE2) to convey trust and intelligence.
- Background color: Light desaturated blue (#D1E9F5), complementing the primary blue, creating a calming effect.
- Accent color: Analogous light green (#82CAFA), for interactive elements.
- Font: 'Inter', a grotesque-style sans-serif, for headlines and body text.
- Simple, modern icons to represent skills and features.
- Clean and intuitive layout for easy navigation and user experience.
- Subtle transitions and animations to enhance user interaction.
